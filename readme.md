# 🚀 Sales Management System

![SAPUI5](https://img.shields.io/badge/SAP-UI5-blue)

Enterprise Sales Management System built using SAP CAP and SAPUI5.

This project provides complete management of employees, customers, products and sales with a modern dashboard, authentication and inventory management.

---

## Features

✔ Employee Login Authentication

✔ Dashboard Analytics

✔ Employee Management

✔ Customer Management

✔ Product Inventory

✔ Sales Management

✔ Revenue Calculation

✔ Low Stock Detection

✔ Search & Filter

✔ CRUD Operations

✔ Responsive SAPUI5 Interface
